80x80 Thumbnail Logo

How to Save as 80x80 PNG:

  1. Right-click on the orange square logo above
  2. Select "Inspect" or "Inspect Element"
  3. Right-click the highlighted <div class="logo-80"> in DevTools
  4. Select "Capture node screenshot"
  5. OR: Use Snipping Tool to capture just the orange square (80x80)
  6. Save as thumbnail.png

How to Save as 590x300 PNG:

  1. Zoom out browser to 100% (Ctrl+0 or Cmd+0)
  2. Use Snipping Tool to capture the orange banner above
  3. Make sure it captures exactly 590x300 pixels
  4. Save as preview.png
